# ystock

It's a Chrome extension for easily displaying stock information of Taiwan.
All of the information is grabbed from [Yahoo!奇摩股市](https://tw.stock.yahoo.com/).

這是一個可以簡單的看台灣股票的小工具
資料是由 [Yahoo!奇摩股市](https://tw.stock.yahoo.com/) 取得
